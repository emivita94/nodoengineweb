Guía de integración NODO Engine

API REST para facturación electrónica Paraguay · SET / SIFEN · v1

Inicio rápido — de cero a primera factura

1
Obtener las credenciales
En el panel NODO Engine → empresa → Integración ERP encontrás dos datos:
• URL base — la dirección del servidor de la API
• API Key — la clave secreta que identifica a la empresa emisora
Guardá estos dos valores en la configuración de tu sistema. No los pongas hardcodeados en el código fuente. 
' Variables de configuración recomendadas (VB.NET)
' En app.config, web.config, o tabla de parámetros de tu BD
Dim NODO_URL As String = ConfigurationManager.AppSettings("NodoApiUrl")
Dim NODO_KEY As String = ConfigurationManager.AppSettings("NodoApiKey")
2
Verificar que la API está disponible
Antes de integrar, verificá que tu sistema puede alcanzar el servidor. Este endpoint no requiere API Key.
GET /health 
' VB.NET — verificar conectividad
Dim client As New HttpClient()
Dim response As HttpResponseMessage = Await client.GetAsync(NODO_URL.Replace("/api/v1", "") & "/health")
Dim json As String = Await response.Content.ReadAsStringAsync()
' Respuesta esperada: {"ok":true,"version":"0.1.0","ambiente":"prod"}
3
Emitir la primera factura de prueba
Usá una API Key que empiece con sk_test_ para probar — los documentos de test no van a la SET real.
El cliente innominado (tipo: 4) es el más simple porque no requiere RUC ni CI. 
' VB.NET — factura mínima de prueba
Dim json As String = "{""tipoDocumento"":1,""cliente"":{""tipo"":4},""items"":[{""descripcion"":""Prueba"",""cantidad"":1,""precioUnitario"":100000,""precioTotal"":100000,""tasaIVA"":10}],""referenciaExterna"":""PRUEBA-001""}"
Dim result As JObject = Await NodoPost("/documentos/erp", json)
MessageBox.Show("Estado: " & result("data")("estado").ToString())
4
Guardar el CDC y verificar el resultado
Si el documento fue aprobado, la respuesta tiene un cdc de 44 dígitos. Guardalo en tu base de datos — es el identificador oficial ante la SET. 
' Estructura de respuesta exitosa
{
  "ok": true,
  "data": {
    "cdc":           "01803778760001002000001...",  // ← GUARDAR EN BD (44 dígitos)
    "numero":        "001-002-0000001",            // Número visible en el KUDE
    "estado":        "aprobado",                   // aprobado | rechazado | pendiente
    "sifen_codigo":  "0260",                       // 0260 = SET aprobó
    "sifen_mensaje": "Documento aprobado"
  }
}
✅ Si llegaste hasta acá, la integración básica está funcionando. Navegá por el menú de la izquierda para ver todos los endpoints en detalle, el manejo de errores y los ejemplos completos en VB.NET.

Autenticación

Todos los endpoints (excepto /health) requieren una API Key válida. La clave se envía como header HTTP en cada request.

Header obligatorio

X-API-Key: sk_live_XXXXXXXXXXXXXXXX
⚠️ La API Key es secreta. No la incluyas en logs, no la expongas en el frontend web, no la hardcodees en el código fuente. Guardala en variables de entorno, app.config cifrado, o una tabla de parámetros de tu base de datos.

Tipos de API Key

PrefijoAmbienteDescripción
sk_test_Test / PruebasLos documentos se validan localmente pero NO se envían a la SET. Usar para desarrollo y pruebas de integración.
sk_live_ProducciónLos documentos se envían a la SET real y quedan registrados oficialmente. Usar solo cuando la integración esté probada.

Implementación en VB.NET

Lo más práctico es crear una clase helper que maneje todas las llamadas a la API:

' NodoApiClient.vb — Clase helper para llamadas a NODO Engine
Imports System.Net.Http
Imports System.Net.Http.Headers
Imports Newtonsoft.Json.Linq

Public Class NodoApiClient

    Private Shared ReadOnly _http As New HttpClient()
    Private Shared _baseUrl As String
    Private Shared _apiKey  As String

    Public Shared Sub Inicializar(baseUrl As String, apiKey As String)
        _baseUrl = baseUrl.TrimEnd("/"c)
        _apiKey  = apiKey
        _http.DefaultRequestHeaders.Clear()
        _http.DefaultRequestHeaders.Add("X-API-Key", _apiKey)
        _http.Timeout = TimeSpan.FromSeconds(30)
    End Sub

    ' POST — envía JSON y retorna el objeto "data" de la respuesta
    Public Shared Async Function Post(endpoint As String, payload As Object) As Task(Of JObject)
        Dim json    As String          = Newtonsoft.Json.JsonConvert.SerializeObject(payload)
        Dim content As New StringContent(json, System.Text.Encoding.UTF8, "application/json")
        Dim res     As HttpResponseMessage = Await _http.PostAsync(_baseUrl & endpoint, content)
        Return Await ParseResponse(res)
    End Function

    ' GET — retorna el objeto "data"
    Public Shared Async Function [Get](endpoint As String) As Task(Of JObject)
        Dim res As HttpResponseMessage = Await _http.GetAsync(_baseUrl & endpoint)
        Return Await ParseResponse(res)
    End Function

    ' GET binario — para descargar PDF/XML
    Public Shared Async Function GetBytes(endpoint As String) As Task(Of Byte())
        Dim res As HttpResponseMessage = Await _http.GetAsync(_baseUrl & endpoint)
        If Not res.IsSuccessStatusCode Then
            Throw New Exception("Error " & res.StatusCode.ToString())
        End If
        Return Await res.Content.ReadAsByteArrayAsync()
    End Function

    Private Shared Async Function ParseResponse(res As HttpResponseMessage) As Task(Of JObject)
        Dim body As String = Await res.Content.ReadAsStringAsync()
        Dim obj  As JObject = JObject.Parse(body)

        ' Si hay error HTTP (4xx, 5xx)
        If Not res.IsSuccessStatusCode Then
            Dim msg As String = obj("error")?.ToString() ?? res.StatusCode.ToString()
            Throw New Exception("Error API: " & msg)
        End If

        Return obj
    End Function

End Class

' Inicializar al arrancar la aplicación (Form_Load o Sub Main):
NodoApiClient.Inicializar(
    ConfigurationManager.AppSettings("NodoApiUrl"),
    ConfigurationManager.AppSettings("NodoApiKey")
)
Dependencia requerida: esta clase usa Newtonsoft.Json para manejar JSON. Instalala vía NuGet: Install-Package Newtonsoft.Json. También podés usar System.Text.Json si preferís no agregar dependencias.

Emitir una factura electrónica

Este es el endpoint principal. Recibe los datos de la transacción, genera el XML según la norma SIFEN, lo firma digitalmente con el certificado del emisor y lo envía a la SET — todo en una sola llamada sincrónica.

POST /api/v1/documentos/erp

Estructura del request

{
  // ── TIPO DE DOCUMENTO ──────────────────────────────────────────────────────
  "tipoDocumento": 1,              // 1=Factura, 4=Autofactura, 5=NC, 6=ND, 7=Remisión

  // ── FECHA Y CONDICIONES ─────────────────────────────────────────────────────
  "fecha": "2025-04-03T10:30:00",  // ISO 8601. Si se omite, se usa la hora actual
  "moneda": "PYG",                 // PYG (defecto) | USD | BRL | EUR | ARS
  "condicionVenta": 1,             // 1=Contado (defecto) | 2=Crédito

  // ── RECEPTOR ────────────────────────────────────────────────────────────────
  "cliente": {
    "tipo": 1,                      // 1=RUC | 2=CI | 3=Pasaporte/extranjero | 4=Innominado
    "documento": "80012345-6",     // RUC con dígito verificador. No enviar si tipo=4
    "razonSocial": "Mi Empresa SA", // Nombre completo. No enviar si tipo=4
    "email": "fe@empresa.py",       // Opcional — NODO envía el KUDE por email automáticamente
    "telefono": "021555000",        // Opcional
    "direccion": "Av. Principal 123" // Opcional
  },

  // ── LÍNEAS DE DETALLE ────────────────────────────────────────────────────────
  "items": [
    {
      "descripcion": "Notebook HP 15",  // Descripción del bien o servicio
      "cantidad": 2,                     // Puede ser decimal: 1.5
      "precioUnitario": 2500000,          // Sin IVA incluido. Sin decimales en PYG.
      "precioTotal": 5000000,            // cantidad × precioUnitario (el motor valida coherencia)
      "tasaIVA": 10,                     // 0=Exento | 5=IVA 5% | 10=IVA 10%
      "unidadMedida": 77,                // 77=Unidad (defecto) | 83=Servicio | ver tabla
      "codigoProducto": "NB-HP-15"      // Opcional — código interno del ERP
    },
    {
      "descripcion": "Soporte técnico",
      "cantidad": 1,
      "precioUnitario": 500000,
      "precioTotal": 500000,
      "tasaIVA": 0                      // Exento de IVA — mismo comprobante puede tener distintas tasas
    }
  ],

  // ── TRAZABILIDAD ─────────────────────────────────────────────────────────────
  "referenciaExterna": "FAC-2025-0042", // Tu ID interno — se devuelve en el webhook
  "webhookUrl": "https://mi-erp.py/webhook/nodo" // Opcional — NODO notifica aquí cuando la SET responde
}

Ejemplo completo en VB.NET

Public Async Function EmitirFactura(venta As VentaDTO) As Task(Of ResultadoNodo)
    ' Construir el objeto a enviar
    Dim payload = New With {
        .tipoDocumento  = 1,
        .fecha          = venta.Fecha.ToString("yyyy-MM-ddTHH:mm:ss"),
        .moneda         = "PYG",
        .condicionVenta = If(venta.EsCredito, 2, 1),
        .cliente = New With {
            .tipo       = venta.TipoDocCliente,  ' 1=RUC, 2=CI, 4=Sin doc
            .documento  = venta.RucCiCliente,
            .razonSocial = venta.NombreCliente,
            .email      = venta.EmailCliente      ' NODO envía el KUDE por email
        },
        .items = venta.Detalles.Select(Function(d) New With {
            .descripcion    = d.Descripcion,
            .cantidad       = d.Cantidad,
            .precioUnitario = d.PrecioUnitario,
            .precioTotal    = d.PrecioTotal,
            .tasaIVA        = d.TasaIVA,          ' 0, 5 ó 10
            .codigoProducto = d.CodigoInterno
        }).ToArray(),
        .referenciaExterna = venta.NroFacturaInterno,
        .webhookUrl        = ConfigurationManager.AppSettings("NodoWebhookUrl")
    }

    Try
        Dim respuesta = Await NodoApiClient.Post("/documentos/erp", payload)
        Dim data      = respuesta("data")

        ' Guardar el CDC en tu BD SIEMPRE — aunque sea rechazado
        Dim resultado As New ResultadoNodo With {
            .CDC           = data("cdc").ToString(),
            .Numero        = data("numero").ToString(),
            .Estado        = data("estado").ToString(),
            .SifenCodigo   = data("sifen_codigo").ToString(),
            .SifenMensaje  = data("sifen_mensaje").ToString()
        }

        ' Actualizar en tu BD
        FacturasDao.ActualizarCDC(venta.NroFacturaInterno, resultado.CDC, resultado.Estado)

        If resultado.Estado = "aprobado" Then
            ' Descargar y guardar el KUDE automáticamente
            Await DescargarYGuardarKude(resultado.CDC, venta.NroFacturaInterno)
        Else
            ' Documento rechazado — mostrar el motivo al usuario
            MessageBox.Show("SET rechazó el documento: " & resultado.SifenMensaje,
                           "Factura rechazada", MessageBoxButtons.OK, MessageBoxIcon.Warning)
        End If

        Return resultado

    Catch ex As Exception
        ' Error de red o de la API (4xx, 5xx)
        LogError(ex.Message)
        MessageBox.Show("Error al enviar factura: " & ex.Message)
        Return Nothing
    End Try
End Function
Importante sobre documentos rechazados: cuando la SET rechaza un documento, la API igualmente devuelve HTTP 201 con estado: "rechazado". El documento queda registrado con un número secuencial asignado. Si necesitás corregirlo, debés crear un documento nuevo (con nuevo número) — no se puede reenviar el mismo CDC.

Emitir una nota de crédito

La nota de crédito ajusta o anula parcial o totalmente una factura aprobada. Usa el mismo endpoint que la factura, con tipoDocumento: 5 y referenciando la factura original con su CDC.

POST /api/v1/documentos/erp

Request

  "fecha": "2025-04-05T09:00:00",
  "moneda": "PYG",

  // OBLIGATORIO: referencia a la factura original
  "documentoAsociado": {

Ejemplo en VB.NET

Public Async Function EmitirNotaCredito(cdcFacturaOriginal As String, devolucion As DevolucionDTO) As Task(Of ResultadoNodo)
    Dim payload = New With {
        .tipoDocumento = 5,
        .fecha         = DateTime.Now.ToString("yyyy-MM-ddTHH:mm:ss"),
        .moneda        = "PYG",
        .documentoAsociado = New With {
            .cdc                    = cdcFacturaOriginal,
            .tipoDocumentoAsociado  = 1
        },
        .motivoEmision = devolucion.Motivo,  ' 1=Devolución, 2=Anulación, etc.
        .cliente = New With {
            .tipo        = devolucion.TipoDocCliente,
            .documento   = devolucion.RucCiCliente,
            .razonSocial = devolucion.NombreCliente
        },
        .items = devolucion.Items.Select(Function(i) New With {
            .descripcion    = "Devolución: " & i.Descripcion,
            .cantidad       = i.CantidadDevuelta,
            .precioUnitario = i.PrecioUnitario,
            .precioTotal    = i.PrecioUnitario * i.CantidadDevuelta,
            .tasaIVA        = i.TasaIVA
        }).ToArray(),
        .referenciaExterna = devolucion.NroInternoNC
    }

    Return Await EnviarDocumento(payload)
End Function

Cancelar un documento aprobado

Solo se pueden cancelar documentos con estado: "aprobado". La cancelación se registra en la SET.

POST /api/v1/documentos/{cdc}/cancelar 
// Body:
{ "motivo": "Error en datos del cliente" }

// Respuesta:
{ "ok": true, "data": { "cdc": "...", "estado": "cancelado" } }

VB.NET

Public Async Function CancelarDocumento(cdc As String, motivo As String) As Task(Of Boolean)
    Try
        Dim payload = New With { .motivo = motivo }
        Dim resp    = Await NodoApiClient.Post("/documentos/" & cdc & "/cancelar", payload)
        Return resp("ok").Value(Of Boolean)()
    Catch ex As Exception
        MessageBox.Show("Error cancelando: " & ex.Message)
        Return False
    End Try
End Function

Descargar KUDE (PDF) y XML firmado

Solo disponible para documentos con estado: "aprobado". Ambos endpoints devuelven el archivo binario directamente.

⚠️ Estos endpoints no son URLs públicas. Requieren el header X-API-Key igual que el resto. No funcionan abriéndolos directo en el browser sin autenticación.

Descargar KUDE en PDF

GET /api/v1/documentos/{cdc}/kude?formato=a4

Parámetro formato: a4 (defecto) · carta · ticket

Descargar XML firmado

GET /api/v1/documentos/{cdc}/xml

VB.NET — descargar KUDE y guardarlo o mostrarlo

' Opción 1: Guardar el PDF en disco
Public Async Sub GuardarKudeEnDisco(cdc As String, rutaArchivo As String)
    Dim bytes() As Byte = Await NodoApiClient.GetBytes("/documentos/" & cdc & "/kude?formato=a4")
    File.WriteAllBytes(rutaArchivo, bytes)
    ' Ej: "C:\Facturas\KUDE-001-002-0000001.pdf"
End Sub

' Opción 2: Abrir el PDF en el visor predeterminado del sistema
Public Async Sub AbrirKude(cdc As String)
    Dim bytes()  As Byte  = Await NodoApiClient.GetBytes("/documentos/" & cdc & "/kude?formato=a4")
    Dim tempPath As String = Path.Combine(Path.GetTempPath(), cdc & ".pdf")
    File.WriteAllBytes(tempPath, bytes)
    Process.Start(tempPath)  ' Abre con Adobe Reader, Foxit, etc.
End Sub

' Opción 3: Mostrar en un PictureBox o WebBrowser dentro del formulario
Public Async Sub MostrarKudeEnFormulario(cdc As String, viewer As WebBrowser)
    Dim bytes()  As Byte  = Await NodoApiClient.GetBytes("/documentos/" & cdc & "/kude?formato=a4")
    Dim tempPath As String = Path.Combine(Path.GetTempPath(), cdc & ".pdf")
    File.WriteAllBytes(tempPath, bytes)
    viewer.Navigate(tempPath)
End Sub

' Opción 4: Enviar el PDF por email desde tu sistema
Public Async Sub EnviarKudePorEmail(cdc As String, destinatario As String)
    Dim bytes() As Byte = Await NodoApiClient.GetBytes("/documentos/" & cdc & "/kude?formato=a4")
    Dim mail    As New MailMessage()
    mail.To.Add(destinatario)
    mail.Subject    = "Factura electrónica"
    mail.Attachments.Add(New Attachment(New MemoryStream(bytes), "factura.pdf", "application/pdf"))
    ' ... enviar con SmtpClient
End Sub
Tip: si el campo cliente.email se incluyó al emitir el documento, NODO Engine envía automáticamente el KUDE por email al cliente al momento de la aprobación. No necesitás descargarlo y reenviarlo manualmente.

Si al emitir el documento incluís el campo webhookUrl, NODO hace un POST a esa URL cuando la SET responde. Así tu sistema se actualiza en tiempo real sin tener que hacer polling.

Implementar el endpoint webhook en VB.NET (ASP.NET WebAPI)

' WebhookController.vb
 WebhookController
    Inherits ApiController

    
    "webhook/nodo")>
    Public Async Function RecibirWebhook(payload As JObject) As Task(Of IHttpActionResult)

        ' PASO 1: Responder 200 INMEDIATAMENTE (antes de procesar)
        ' NODO considera el webhook como fallido si no responde en 10 segundos
        Dim response = Ok()
        Task.Run(Sub() ProcesarWebhookAsync(payload))  ' Procesar en background
        Return response

    End Function

    Private Async Sub ProcesarWebhookAsync(payload As JObject)
        Try
            Dim evento     As String = payload("evento").ToString()
            Dim documento           = payload("documento")
            Dim cdc        As String = documento("cdc").ToString()
            Dim refExterna As String = documento("referenciaExterna")?.ToString()
            Dim estado     As String = documento("estado").ToString()

            Select Case evento
                Case "de.aprobado"
                    ' Actualizar estado en tu BD usando la referencia interna
                    ' IMPORTANTE: usar refExterna como clave para evitar duplicados
                    FacturasDao.ActualizarEstado(refExterna, cdc, "APROBADA")

                Case "de.rechazado"
                    Dim motivo = documento("sifen")("mensaje").ToString()
                    FacturasDao.ActualizarEstado(refExterna, cdc, "RECHAZADA", motivo)
                    ' Notificar al usuario que debe corregir y reemitir

                Case "de.cancelado"
                    FacturasDao.ActualizarEstado(refExterna, cdc, "CANCELADA")
            End Select

        Catch ex As Exception
            LogError("Error procesando webhook: " & ex.Message)
        End Try
    End Sub

End Class

Cálculo del IVA

' El motor calcula el IVA automáticamente. Solo enviás precioTotal.
' NO es necesario ni correcto enviar el monto del IVA.

' Fórmula que usa el motor internamente:
IVA 10% = precioTotal / 11
IVA  5% = precioTotal / 21
Exento  = 0

' Ejemplo en VB.NET (para mostrar en tu UI):
Function CalcularIVA(precioTotal As Decimal, tasaIVA As Integer) As Decimal
    Select Case tasaIVA
        Case 10 : Return Math.Round(precioTotal / 11, 0)
        Case  5 : Return Math.Round(precioTotal / 21, 0)
        Case  0 : Return 0
        Case Else : Throw New ArgumentException("tasaIVA debe ser 0, 5 ó 10")
    End Select
End Function

Clase NodoApiClient completa (lista para usar)

Copiá esta clase en tu proyecto. Requiere el paquete NuGet Newtonsoft.Json.

' ============================================================
' NodoApiClient.vb — Cliente para NODO Engine API
' Requiere: Newtonsoft.Json (NuGet)
' ============================================================
Imports System.Net.Http
Imports Newtonsoft.Json
Imports Newtonsoft.Json.Linq

Public Class NodoApiClient

    Private Shared ReadOnly _http     As New HttpClient() With { .Timeout = TimeSpan.FromSeconds(60) }
    Private Shared          _baseUrl  As String
    Private Shared          _apiKey   As String
    Private Shared          _listo    As Boolean = False

    Public Shared Sub Inicializar(baseUrl As String, apiKey As String)
        _baseUrl = baseUrl.TrimEnd("/"c)
        _apiKey  = apiKey
        _http.DefaultRequestHeaders.Remove("X-API-Key")
        _http.DefaultRequestHeaders.Add("X-API-Key", _apiKey)
        _listo = True
    End Sub

    ' ── Verificar conectividad ──────────────────────────────────
    Public Shared Async Function VerificarConexion() As Task(Of Boolean)
        Try
            Dim url = _baseUrl.Replace("/api/v1", "") & "/health"
            Dim res = Await _http.GetAsync(url)
            Return res.IsSuccessStatusCode
        Catch
            Return False
        End Try
    End Function

    ' ── Emitir documento (factura, NC, etc.) ────────────────────
    Public Shared Async Function EmitirDocumento(payload As Object) As Task(Of ResultadoNodo)
        Dim resp = Await Post("/documentos/erp", payload)
        Dim d    = resp("data")
        Return New ResultadoNodo With {
            .CDC          = d("cdc").ToString(),
            .Numero       = d("numero").ToString(),
            .Estado       = d("estado").ToString(),
            .SifenCodigo  = d("sifen_codigo")?.ToString(),
            .SifenMensaje = d("sifen_mensaje")?.ToString(),
            .Aprobado     = d("estado").ToString() = "aprobado"
        }
    End Function

    ' ── Consultar estado ────────────────────────────────────────
    Public Shared Async Function ConsultarPorCDC(cdc As String) As Task(Of ResultadoNodo)
        Dim resp = Await [Get]("/documentos/" & cdc)
        Dim d    = resp("data")
        Return New ResultadoNodo With {
            .CDC          = d("cdc").ToString(),
            .Estado       = d("estado").ToString(),
            .SifenMensaje = d("sifen_mensaje")?.ToString(),
            .Aprobado     = d("estado").ToString() = "aprobado"
        }
    End Function

    ' ── Descargar KUDE en bytes ─────────────────────────────────
    Public Shared Async Function DescargarKude(cdc As String, Optional formato As String = "a4") As Task(Of Byte())
        Return Await GetBytes($"/documentos/{cdc}/kude?formato={formato}")
    End Function

    ' ── Descargar XML en bytes ──────────────────────────────────
    Public Shared Async Function DescargarXML(cdc As String) As Task(Of Byte())
        Return Await GetBytes($"/documentos/{cdc}/xml")
    End Function

    ' ── Cancelar documento ──────────────────────────────────────
    Public Shared Async Function Cancelar(cdc As String, motivo As String) As Task(Of Boolean)
        Dim resp = Await Post($"/documentos/{cdc}/cancelar", New With { .motivo = motivo })
        Return resp("ok").Value(Of Boolean)()
    End Function

    ' ── Métodos HTTP internos ───────────────────────────────────
    Private Shared Async Function Post(endpoint As String, payload As Object) As Task(Of JObject)
        Dim json    = JsonConvert.SerializeObject(payload)
        Dim content = New StringContent(json, System.Text.Encoding.UTF8, "application/json")
        Dim res     = Await _http.PostAsync(_baseUrl & endpoint, content)
        Return Await ParseResponse(res)
    End Function

    Private Shared Async Function [Get](endpoint As String) As Task(Of JObject)
        Dim res = Await _http.GetAsync(_baseUrl & endpoint)
        Return Await ParseResponse(res)
    End Function

    Private Shared Async Function GetBytes(endpoint As String) As Task(Of Byte())
        Dim res = Await _http.GetAsync(_baseUrl & endpoint)
        If Not res.IsSuccessStatusCode Then
            Throw New Exception("Error descargando: HTTP " & CInt(res.StatusCode))
        End If
        Return Await res.Content.ReadAsByteArrayAsync()
    End Function

    Private Shared Async Function ParseResponse(res As HttpResponseMessage) As Task(Of JObject)
        Dim body = Await res.Content.ReadAsStringAsync()
        Dim obj  = JObject.Parse(body)
        If Not res.IsSuccessStatusCode Then
            Throw New Exception("Error API: " & (obj("error")?.ToString() ?? res.StatusCode.ToString()))
        End If
        Return obj
    End Function

End Class

' ── Clase ResultadoNodo ─────────────────────────────────────
Public Class ResultadoNodo
    Public Property CDC          As String
    Public Property Numero       As String
    Public Property Estado       As String
    Public Property SifenCodigo  As String
    Public Property SifenMensaje As String
    Public Property Aprobado     As Boolean
End Class

Python

import requests, os

NODO_URL = os.getenv('NODO_API_URL')
HEADERS  = {'X-API-Key': os.getenv('NODO_API_KEY'), 'Content-Type': 'application/json'}

def emitir_factura(cliente, items, referencia):
    res = requests.post(f'{NODO_URL}/documentos/erp',
        json={'tipoDocumento': 1, 'cliente': cliente, 'items': items, 'referenciaExterna': referencia},
        headers=HEADERS, timeout=30)
    res.raise_for_status()
    return res.json()['data']

def descargar_kude(cdc, ruta):
    res = requests.get(f'{NODO_URL}/documentos/{cdc}/kude?formato=a4', headers=HEADERS)
    res.raise_for_status()
    with open(ruta, 'wb') as f: f.write(res.content)

PHP

function nodo_post($endpoint, $data) {
    $ch = curl_init($_ENV['NODO_API_URL'] . $endpoint);
    curl_setopt_array($ch, [
        CURLOPT_POST           => true,
        CURLOPT_POSTFIELDS     => json_encode($data),
        CURLOPT_RETURNTRANSFER => true,
        CURLOPT_TIMEOUT        => 30,
        CURLOPT_HTTPHEADER     => ['Content-Type: application/json', 'X-API-Key: ' . $_ENV['NODO_API_KEY']],
    ]);
    $res = json_decode(curl_exec($ch), true);
    curl_close($ch);
    if (!$res['ok']) throw new Exception($res['error']);
    return $res['data'];
}
