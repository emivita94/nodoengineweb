Guia de integracion NODO Engine

API REST -- Facturacion electronica Paraguay -- SET / SIFEN -- v1

Esta guia te explica paso a paso como conectar tu sistema (ya sea una aplicacion de escritorio en VB.NET o un punto de venta web en JavaScript) con la API de NODO Engine para emitir facturas electronicas en Paraguay.

Que es NODO Engine? Es un servicio que se encarga de todo lo complicado: genera el XML segun las reglas de la SET, lo firma con tu certificado digital, lo envia al SIFEN y te devuelve el resultado. Vos solo mandas los datos de la factura en formato JSON y listo.

De cero a primera factura en 5 pasos

1
Obtener las credenciales

En el panel NODO Engine, anda a tu empresa y busca la seccion Integracion ERP. Ahi vas a encontrar dos cosas que necesitas:

  • URL base: la direccion donde esta la API (ejemplo: https://sifen-engine-production-f96e.up.railway.app/api/v1)
  • API Key: una clave secreta que identifica a tu empresa (empieza con sk_test_ o sk_live_)
' Guardar en app.config (NUNCA directamente en el codigo fuente)
' <appSettings>
'   <add key="NodoApiUrl" value="https://sifen-engine-production-f96e.up.railway.app/api/v1"/>
'   <add key="NodoApiKey" value="sk_test_XXXX"/>
' </appSettings>

Dim NODO_URL As String = ConfigurationManager.AppSettings("NodoApiUrl")
Dim NODO_KEY As String = ConfigurationManager.AppSettings("NodoApiKey")
// Guardar en variables de configuracion, no en el codigo
const NODO_URL = 'https://sifen-engine-production-f96e.up.railway.app/api/v1';
const NODO_KEY = 'sk_test_XXXX';
2
Verificar que la API esta funcionando

Antes de hacer nada, verifica que la API responda. Este endpoint no requiere API Key -- es publico.

GET/health
// Si todo esta bien, responde:
{
  "ok": true,
  "version": "1.0.0",
  "ambiente": "prod"
}
3
Validar los datos antes de emitir (opcional pero recomendado)

Antes de emitir de verdad, podes enviar los mismos datos al endpoint de validacion. Esto te dice si hay errores sin enviar nada a la SET. Es como un "ensayo".

POST/api/v1/documentos/validar

Autenticacion

Todos los endpoints (excepto /health) necesitan que envies tu API Key en un header HTTP. Sin este header, la API te va a responder con error 401.

Que header enviar en cada request?

En cada llamada a la API, tenes que incluir estos dos headers:

Content-Type: application/json
X-API-Key: sk_live_XXXXXXXXXXXXXXXX
La API Key es secreta. No la pongas directamente en tu codigo fuente, no la muestres en logs ni la compartas. Guardala en un archivo de configuracion (app.config en VB.NET) o en una variable de entorno.

Tipos de API Key

PrefijoAmbientePara que sirve
sk_test_TestPara desarrollo y pruebas. Los documentos no se envian a la SET. Usa esta mientras estas programando.
sk_live_ProduccionDocumentos reales que quedan registrados en la SET. Solo usar cuando tu integracion este probada y funcionando.

Como se ve en codigo?

' En tu app.config:
' <appSettings>
'   <add key="NodoApiUrl" value="https://sifen-engine-production-f96e.up.railway.app/api/v1"/>
'   <add key="NodoApiKey" value="sk_live_XXXXXXXXXXXXXXXX"/>
' </appSettings>

' En tu codigo, leer asi:
Dim baseUrl As String = ConfigurationManager.AppSettings("NodoApiUrl")
Dim apiKey  As String = ConfigurationManager.AppSettings("NodoApiKey")

' Configurar el HttpClient con el header:
Dim http As New HttpClient()
http.DefaultRequestHeaders.Add("X-API-Key", apiKey)
// Guardar en configuracion, no en el codigo
const BASE_URL = 'https://sifen-engine-production-f96e.up.railway.app/api/v1';
const API_KEY  = 'sk_live_XXXXXXXXXXXXXXXX';

// En cada fetch, incluir los headers:
const headers = {
  'Content-Type': 'application/json',
  'X-API-Key': API_KEY
};

Validar antes de emitir NUEVO

Este endpoint te permite verificar que los datos de un documento estan correctos antes de emitirlo de verdad. Es como hacer un "ensayo" -- la API revisa todos los campos y te dice si hay errores, pero no envia nada a la SET ni genera ningun documento.

Cuando usar esto? Es muy util para validar los datos en tiempo real mientras el usuario completa el formulario de facturacion, o para hacer una verificacion final antes de confirmar la emision. Asi evitas errores que despues son dificiles de corregir.
POST/api/v1/documentos/validar

Headers requeridos

Content-Type: application/json
X-API-Key: sk_live_XXXX

Body (es el mismo que usarias para emitir)

Envias exactamente el mismo JSON que enviarias al endpoint /documentos/erp. La API lo valida campo por campo.

{
  "tipoDocumento": 1,
  "cliente": {
    "tipo": 1,
    "documento": "80012345-6",
    "razonSocial": "Mi Empresa SA"
  },
  "items": [{
    "descripcion": "Notebook HP 15",
    "cantidad": 2,
    "precioUnitario": 2500000,
    "precioTotal": 5000000,
    "tasaIVA": 10
  }],
  "referenciaExterna": "FAC-2025-0042"
}

Respuesta exitosa (todo esta bien)

// HTTP 200
{
  "ok": true,
  "valido": true,
  "errores": [],
  "resumen": {
    "tipoDocumento": "Factura electronica",
    "timbrado": "12345678",
    "establecimiento": "001",
    "punto": "002",
    "ambiente": "test",
    "montoTotal": 5000000,
    "receptor": "Mi Empresa SA (80012345-6)"
  }
}

Respuesta con errores

// HTTP 200 (no es un error HTTP, es una validacion)
{
  "ok": false,
  "valido": false,
  "errores": [
    {
      "campo": "items[0].tasaIVA",
      "mensaje": "Tasa IVA debe ser 0, 5 o 10"
    },
    {
      "campo": "cliente.documento",
      "mensaje": "RUC invalido, verificar formato (ej: 80012345-6)"
    }
  ]
}

Public Async Function ValidarDocumento(payload As Object) As Task(Of Boolean)
  Try
    Dim json = JsonConvert.SerializeObject(payload)
    Dim content = New StringContent(json, Encoding.UTF8, "application/json")
    Dim resp = Await _http.PostAsync(_baseUrl & "/documentos/validar", content)
    Dim body = Await resp.Content.ReadAsStringAsync()
    Dim obj = JObject.Parse(body)

    If obj("valido").Value(Of Boolean)() Then
      MessageBox.Show("Datos correctos. Podes emitir.", "Validacion OK")
      Return True
    Else
      ' Mostrar los errores al usuario
      Dim errores = obj("errores").Select(
        Function(e) e("campo").ToString() & ": " & e("mensaje").ToString()
      )
      MessageBox.Show(String.Join(vbCrLf, errores), "Errores encontrados")
      Return False
    End If
  Catch ex As Exception
    MessageBox.Show("Error al validar: " & ex.Message)
    Return False
  End Try
End Function

' Uso: validar antes de emitir
If Await ValidarDocumento(miPayload) Then
  Dim r = Await NodoApiClient.EmitirDocumento(miPayload)
  ' ... procesar resultado
End If
async function validarDocumento(payload) {
  const resp = await fetch(BASE_URL + '/documentos/validar', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', 'X-API-Key': API_KEY },
    body: JSON.stringify(payload)
  });
  const data = await resp.json();

  if (data.valido) {
    console.log('Datos correctos, se puede emitir');
    console.log('Resumen:', data.resumen);
    return true;
  } else {
    console.error('Errores encontrados:');
    data.errores.forEach(e => {
      console.error(`  ${e.campo}: ${e.mensaje}`);
    });
    return false;
  }
}

// Uso: validar antes de emitir
if (await validarDocumento(miPayload)) {
  const resultado = await emitirDocumento(miPayload);
  // ... procesar resultado
}

Emitir una factura electronica

Este es el endpoint principal. Con una sola llamada, NODO Engine genera el XML segun el formato SIFEN, lo firma con tu certificado digital y lo envia a la SET. Vos solo envias los datos en JSON.

POST/api/v1/documentos/erp

Headers requeridos

Content-Type: application/json
X-API-Key: sk_live_XXXX

Body completo con explicacion de cada campo

{
  // --- TIPO DE DOCUMENTO ---
  "tipoDocumento": 1,              // 1=Factura | 5=Nota credito | 6=Nota debito | 7=Nota remision

  // --- FECHA Y CONDICIONES (opcionales) ---
  "fecha": "2025-04-03T10:30:00",  // Formato ISO 8601. Si no lo envias, usa la hora del servidor
  "moneda": "PYG",                 // PYG (defecto) | USD | BRL | EUR
  "condicionVenta": 1,             // 1=Contado (defecto) | 2=Credito

  // --- DATOS DEL CLIENTE ---
  "cliente": {
    "tipo": 1,                      // 1=RUC | 2=CI | 3=Pasaporte | 4=Innominado (sin documento)
    "documento": "80012345-6",     // RUC con digito verificador. NO enviar si tipo=4
    "razonSocial": "Mi Empresa SA", // Nombre o razon social. NO enviar si tipo=4
    "email": "fe@empresa.py",       // Opcional. NODO envia el KUDE por email al aprobarse
    "telefono": "021555000",        // Opcional
    "direccion": "Av. Principal 123" // Opcional
  },

  // --- ITEMS (minimo 1) ---
  "items": [
    {
      "descripcion": "Notebook HP 15",
      "cantidad": 2,                 // Puede ser decimal: 1.5
      "precioUnitario": 2500000,     // Precio por unidad. Sin decimales si es PYG
      "precioTotal": 5000000,        // = cantidad x precioUnitario
      "tasaIVA": 10,                 // 0=Exento | 5=IVA 5% | 10=IVA 10%
      "unidadMedida": 77,            // 77=Unidad (defecto) | 83=Servicio | ver tabla
      "codigoProducto": "NB-HP-15"  // Opcional. Tu codigo interno
    },
    {
      "descripcion": "Soporte tecnico",
      "cantidad": 1,
      "precioUnitario": 500000,
      "precioTotal": 500000,
      "tasaIVA": 0   // Exento. Podes mezclar tasas en un mismo documento
    }
  ],

  // --- TRAZABILIDAD ---
  "referenciaExterna": "FAC-2025-0042", // Tu ID interno. Vuelve en el webhook
  "webhookUrl": "https://mi-erp.py/webhook/nodo" // Opcional. NODO te avisa cuando la SET responde
}

Respuesta exitosa

// HTTP 201
{
  "ok": true,
  "data": {
    "cdc":           "01803778760001002000001...",
    "numero":        "001-002-0000001",
    "estado":        "aprobado",
    "sifen_codigo":  "0260",
    "sifen_mensaje": "Documento aprobado"
  }
}

Respuesta con rechazo de la SET

// HTTP 201 (ojo: sigue siendo 201, no 4xx)
{
  "ok": true,
  "data": {
    "cdc":           "01803778760001002000001...",
    "numero":        "001-002-0000002",
    "estado":        "rechazado",
    "sifen_codigo":  "0411",
    "sifen_mensaje": "RUC del receptor no valido"
  }
}
Importante: cuando la SET rechaza un documento, el HTTP status sigue siendo 201. Siempre revisa el campo data.estado para saber si fue aprobado o rechazado. No alcanza con solo mirar el codigo HTTP.

Respuesta de error (datos invalidos)

// HTTP 400 o 422
{
  "ok": false,
  "error": "El campo 'items' es obligatorio y debe tener al menos un item"
}

Ejemplos de codigo

Public Async Function EmitirFactura(venta As VentaDTO) As Task(Of ResultadoNodo)
  Dim payload = New With {
    .tipoDocumento = 1,
    .moneda = "PYG",
    .fecha = venta.Fecha.ToString("yyyy-MM-ddTHH:mm:ss"),
    .condicionVenta = If(venta.EsCredito, 2, 1),
    .cliente = New With {
      .tipo = venta.TipoDocCliente,
      .documento = venta.RucCiCliente,
      .razonSocial = venta.NombreCliente,
      .email = venta.EmailCliente
    },
    .items = venta.Detalles.Select(Function(d) New With {
      .descripcion = d.Descripcion,
      .cantidad = d.Cantidad,
      .precioUnitario = d.PrecioUnitario,
      .precioTotal = d.PrecioTotal,
      .tasaIVA = d.TasaIVA,
      .codigoProducto = d.CodigoInterno
    }).ToArray(),
    .referenciaExterna = venta.NroFacturaInterno,
    .webhookUrl = ConfigurationManager.AppSettings("NodoWebhookUrl")
  }

  Try
    Dim r = Await NodoApiClient.EmitirDocumento(payload)

    ' Guardar el CDC en la base de datos SIEMPRE
    FacturasDao.ActualizarCDC(venta.NroFacturaInterno, r.CDC, r.Estado)

    If r.Aprobado Then
      ' Descargar el PDF y abrirlo
      Dim tmp = Path.Combine(Path.GetTempPath(), $"KUDE-{r.Numero}.pdf")
      File.WriteAllBytes(tmp, Await NodoApiClient.DescargarKude(r.CDC))
      Process.Start(tmp) ' Abre en Adobe Reader
    Else
      MessageBox.Show("La SET rechazo el documento: " & r.SifenMensaje,
        "Atencion", MessageBoxButtons.OK, MessageBoxIcon.Warning)
    End If

    Return r
  Catch ex As Exception
    MessageBox.Show("Error al emitir: " & ex.Message)
    Return Nothing
  End Try
End Function
async function emitirFactura(venta) {
  const payload = {
    tipoDocumento: 1,
    moneda: 'PYG',
    condicionVenta: venta.esCredito ? 2 : 1,
    cliente: {
      tipo: venta.tipoDocCliente,
      documento: venta.rucCiCliente,
      razonSocial: venta.nombreCliente,
      email: venta.emailCliente
    },
    items: venta.detalles.map(d => ({
      descripcion: d.descripcion,
      cantidad: d.cantidad,
      precioUnitario: d.precioUnitario,
      precioTotal: d.precioTotal,
      tasaIVA: d.tasaIVA,
      codigoProducto: d.codigoInterno
    })),
    referenciaExterna: venta.nroFacturaInterno
  };

  try {
    const resp = await fetch(BASE_URL + '/documentos/erp', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json', 'X-API-Key': API_KEY },
      body: JSON.stringify(payload)
    });

    const data = await resp.json();

    if (!data.ok) {
      console.error('Error API:', data.error);
      return null;
    }

    // Guardar el CDC en tu base de datos
    guardarCDC(venta.nroFacturaInterno, data.data.cdc, data.data.estado);

    if (data.data.estado === 'aprobado') {
      console.log('Factura aprobada!', data.data.numero);
      // Descargar el KUDE
      const pdfResp = await fetch(
        BASE_URL + `/documentos/${data.data.cdc}/kude?formato=ticket80`,
        { headers: { 'X-API-Key': API_KEY } }
      );
      const blob = await pdfResp.blob();
      // Abrir el PDF en una nueva ventana
      window.open(URL.createObjectURL(blob));
    } else {
      alert('SET rechazo: ' + data.data.sifen_mensaje);
    }

    return data.data;
  } catch (err) {
    console.error('Error de red:', err.message);
    return null;
  }
}
Tip: si incluiste cliente.email, NODO envia el KUDE por email automaticamente cuando la SET aprueba el documento (siempre que tengas SMTP configurado en el panel). No necesitas descargarlo y reenviarlo vos.

Emitir nota de credito

La nota de credito se usa para corregir o anular parcialmente una factura ya emitida. Se envia al mismo endpoint que la factura, pero con tipoDocumento: 5 y referenciando la factura original con su CDC.

POST/api/v1/documentos/erp

Headers requeridos

Content-Type: application/json
X-API-Key: sk_live_XXXX

Body de ejemplo

{
  "tipoDocumento": 5,                          // 5 = Nota de credito
  "moneda": "PYG",

  // Referencia a la factura original (OBLIGATORIO)
  "documentoAsociado": {
    "cdc": "01803778760001002000001...",         // CDC de la factura original
    "tipoDocumentoAsociado": 1                  // 1 = era una factura
  },

  // Motivo de la nota de credito
  "notaCreditoDebito": {
    "motivo": 1                                  // 1=Devolucion | 2=Anulacion | 3=Descuento
  },                                              // 4=Bonificacion | 5=Cred. incobrable
                                                   // 6=Recupero costo | 7=Recupero gasto | 8=Ajuste precio

  // Mismo cliente que la factura original
  "cliente": {
    "tipo": 1,
    "documento": "80012345-6",
    "razonSocial": "Mi Empresa SA"
  },

  // Items que se estan devolviendo/corrigiendo
  "items": [{
    "descripcion": "Devolucion Notebook HP 15",
    "cantidad": 1,
    "precioUnitario": 2500000,
    "precioTotal": 2500000,
    "tasaIVA": 10
  }],

  "referenciaExterna": "NC-2025-0001"
}

Respuesta exitosa

// HTTP 201
{
  "ok": true,
  "data": {
    "cdc": "01803778760001002000002...",
    "numero": "001-002-0000002",

Consultar documentos

Hay dos formas de consultar: buscar un documento especifico por su CDC, o listar varios documentos con filtros.

Consultar un documento por CDC

Si tenes el CDC (el codigo de 44 digitos), podes obtener todos los datos del documento.

GET/api/v1/documentos/{cdc}

Headers requeridos

X-API-Key: sk_live_XXXX

Respuesta

{
  "ok": true,
  "data": {
    "cdc": "01803778760001002000001...",

Listar documentos con filtros

Podes buscar documentos usando diferentes filtros. Todos son opcionales y se combinan entre si.

GET/api/v1/documentos?estado=&desde=&hasta=&ref_ext=&limit=&offset=

Parametros de consulta (query string)

ParametroTipoDescripcionEjemplo
estadostringFiltrar por estadoaprobado, rechazado, pendiente, cancelado
desdestringFecha desde (ISO 8601)2025-04-01
hastastringFecha hasta (ISO 8601)2025-04-30
ref_extstringTu referencia externaFAC-2025-0042
limitnumberCantidad maxima de resultados50 (defecto: 20)
offsetnumberSaltar los primeros N resultados (para paginacion)0

// Pagina 2 (50 por pagina):

Los webhooks son notificaciones automaticas. Cuando emitis un documento y la SET responde (aprobado, rechazado, etc.), NODO puede avisar a tu sistema automaticamente haciendo un POST a la URL que vos indiques.

Como funciona?

1

Cuando emitis un documento, incluye el campo webhookUrl con la URL de tu servidor que va a recibir la notificacion.

Verificacion HMAC (seguridad)

Para asegurarte de que el webhook realmente viene de NODO (y no de alguien que quiere hacerse pasar), podes verificar la firma HMAC que viene en los headers.

Headers de firma

HeaderDescripcionEjemplo
X-Webhook-SignatureFirma HMAC-SHA256 del bodysha256=a1b2c3d4e5f6...
X-Webhook-TimestampTimestamp del momento del envio1712150445

Como verificar la firma

La firma se calcula asi:

// Pseudocodigo:
mensaje = timestamp + "." + raw_body
firma = HMAC-SHA256(mensaje, webhook_secret)
esperado = "sha256=" + firma

// Comparar con el header X-Webhook-Signature
Donde esta el webhook_secret? Lo encontras en el panel NODO Engine, en la seccion de Integracion ERP de tu empresa. Es diferente a la API Key.

Reintentos automaticos

Si tu servidor no responde HTTP 200 (o tarda mas de 10 segundos), NODO reintenta automaticamente con backoff exponencial:

IntentoEspera
1Inmediato
210 segundos
330 segundos
42 minutos
510 minutos
Idempotencia: podes recibir el mismo webhook mas de una vez (por reintentos). Usa referenciaExterna o cdc como clave unica para evitar procesar duplicados en tu base de datos.

Public Async Function RecibirWebhook() As Task(Of IHttpActionResult)
  ' 1. Leer el body crudo para verificar la firma
  Dim rawBody = Await Request.Content.ReadAsStringAsync()
  Dim signature = Request.Headers.GetValues("X-Webhook-Signature").FirstOrDefault()
  Dim timestamp = Request.Headers.GetValues("X-Webhook-Timestamp").FirstOrDefault()

  ' 2. Verificar la firma HMAC
  Dim secret = ConfigurationManager.AppSettings("NodoWebhookSecret")
  Dim mensaje = timestamp & "." & rawBody
  Dim hmac = New HMACSHA256(Encoding.UTF8.GetBytes(secret))
  Dim hash = BitConverter.ToString(
    hmac.ComputeHash(Encoding.UTF8.GetBytes(mensaje))
  ).Replace("-", "").ToLower()
  Dim esperado = "sha256=" & hash

  If signature <> esperado Then
    Return Unauthorized() ' Firma invalida, rechazar
  End If

  ' 3. Procesar el webhook (en background para responder rapido)
  Dim payload = JObject.Parse(rawBody)
  Task.Run(Sub()
    Try
      Dim doc = payload("documento")
      Dim ref = doc("referenciaExterna")?.ToString()
      Dim cdc = doc("cdc").ToString()
      Select Case payload("evento").ToString()
        Case "de.aprobado"   : FacturasDao.ActualizarEstado(ref, cdc, "APROBADA")
        Case "de.rechazado"  : FacturasDao.ActualizarEstado(ref, cdc, "RECHAZADA")
        Case "de.cancelado"  : FacturasDao.ActualizarEstado(ref, cdc, "CANCELADA")
        Case "de.inutilizado": FacturasDao.RegistrarInutilizacion(ref)
      End Select
    Catch ex As Exception
      LogError(ex.Message)
    End Try
  End Sub)

  ' 4. Responder 200 INMEDIATAMENTE
  Return Ok()
End Function
import crypto from 'crypto';

// Endpoint receptor de webhooks (Express)
app.post('/webhook/nodo', express.text({ type: '*/*' }), (req, res) => {
  const rawBody = req.body;
  const signature = req.headers['x-webhook-signature'];
  const timestamp = req.headers['x-webhook-timestamp'];
  const secret = process.env.NODO_WEBHOOK_SECRET;

  // 1. Verificar firma HMAC
  const mensaje = timestamp + '.' + rawBody;
  const expected = 'sha256=' + crypto
    .createHmac('sha256', secret)
    .update(mensaje)
    .digest('hex');

  if (signature !== expected) {
    return res.status(401).send('Firma invalida');
  }

  // 2. Responder 200 inmediatamente
  res.status(200).send('OK');

  // 3. Procesar en background
  const payload = JSON.parse(rawBody);
  const { evento, documento } = payload;

  switch (evento) {
    case 'de.aprobado':
      actualizarEstado(documento.referenciaExterna, documento.cdc, 'APROBADA');
      break;
    case 'de.rechazado':
      actualizarEstado(documento.referenciaExterna, documento.cdc, 'RECHAZADA');
      break;
    case 'de.cancelado':
      actualizarEstado(documento.referenciaExterna, documento.cdc, 'CANCELADA');
      break;
  }
});

function verificarWebhookHMAC(rawBody, signature, timestamp, secret) {
  const mensaje = timestamp + '.' + rawBody;
  const expected = 'sha256=' + crypto
    .createHmac('sha256', secret)
    .update(mensaje)
    .digest('hex');

}

Function CalcularIVA(total As Decimal, tasa As Integer) As Decimal
  Select Case tasa
    Case 10 : Return Math.Round(total / 11, 0)
    Case  5 : Return Math.Round(total / 21, 0)
    Case  0 : Return 0
    Case Else : Throw New ArgumentException("tasaIVA debe ser 0, 5 o 10")
  End Select
End Function

}

Imports System.Net.Http
Imports System.Text
Imports Newtonsoft.Json

Public Class NodoApiClient

  Private Shared ReadOnly _http    As New HttpClient() With {.Timeout = TimeSpan.FromSeconds(60)}
  Private Shared          _baseUrl As String

  ' ===== INICIALIZACION =====
  ' Llamar UNA SOLA VEZ al arrancar la aplicacion
  Public Shared Sub Inicializar(baseUrl As String, apiKey As String)
    _baseUrl = baseUrl.TrimEnd("/"c)
    _http.DefaultRequestHeaders.Remove("X-API-Key")
    _http.DefaultRequestHeaders.Add("X-API-Key", apiKey)
  End Sub

  ' ===== VALIDAR DOCUMENTO (sin emitir) =====
  Public Shared Async Function ValidarDocumento(payload As Object) As Task(Of ValidacionNodo)
    Dim json = JsonConvert.SerializeObject(payload)
    Dim content = New StringContent(json, Encoding.UTF8, "application/json")
    Dim resp = Await _http.PostAsync(_baseUrl & "/documentos/validar", content)
    Dim body = Await resp.Content.ReadAsStringAsync()
    Dim obj = JObject.Parse(body)
    Return New ValidacionNodo With {
      .Valido = obj("valido")?.Value(Of Boolean)(),
      .Errores = If(obj("errores") IsNot Nothing,
        obj("errores").Select(Function(e)
          e("campo").ToString() & ": " & e("mensaje").ToString()
        ).ToList(), New List(Of String)())
    }
  End Function

  ' ===== EMITIR DOCUMENTO =====
  Public Shared Async Function EmitirDocumento(payload As Object) As Task(Of ResultadoNodo)
    Dim d = (Await Post("/documentos/erp", payload))("data")
    Return New ResultadoNodo With {
      .CDC = d("cdc").ToString(),
      .Numero = d("numero").ToString(),
      .Estado = d("estado").ToString(),
      .SifenCodigo = d("sifen_codigo")?.ToString(),
      .SifenMensaje = d("sifen_mensaje")?.ToString(),
      .Aprobado = d("estado").ToString() = "aprobado"
    }
  End Function

  ' ===== EMITIR FACTURA (atajo) =====
  Public Shared Async Function EmitirFactura(
    cliente As Object, items As Object(),
    Optional refExterna As String = Nothing,
    Optional webhookUrl As String = Nothing,
    Optional condicion As Integer = 1
  ) As Task(Of ResultadoNodo)
    Return Await EmitirDocumento(New With {
      .tipoDocumento = 1, .moneda = "PYG",
      .condicionVenta = condicion,
      .cliente = cliente, .items = items,
      .referenciaExterna = refExterna,
      .webhookUrl = webhookUrl
    })
  End Function

  ' ===== EMITIR NOTA DE CREDITO =====
  Public Shared Async Function EmitirNotaCredito(
    cdcOriginal As String, motivo As Integer,
    cliente As Object, items As Object(),
    Optional refExterna As String = Nothing
  ) As Task(Of ResultadoNodo)
    Return Await EmitirDocumento(New With {
      .tipoDocumento = 5, .moneda = "PYG",
      .documentoAsociado = New With {.cdc = cdcOriginal, .tipoDocumentoAsociado = 1},
      .notaCreditoDebito = New With {.motivo = motivo},
      .cliente = cliente, .items = items,
      .referenciaExterna = refExterna
    })
  End Function

  ' ===== CONSULTAR POR CDC =====
  Public Shared Async Function ConsultarPorCDC(cdc As String) As Task(Of ResultadoNodo)
    Dim d = (Await [Get]("/documentos/" & cdc))("data")
    Return New ResultadoNodo With {
      .CDC = d("cdc").ToString(),
      .Estado = d("estado").ToString(),
      .SifenMensaje = d("sifen_mensaje")?.ToString(),
      .Aprobado = d("estado").ToString() = "aprobado"
    }
  End Function

  ' ===== CONSULTAR POR REFERENCIA EXTERNA =====
  Public Shared Async Function ConsultarPorReferencia(refExterna As String) As Task(Of ResultadoNodo)
    Dim obj = Await [Get]("/documentos?ref_ext=" & refExterna)
    Dim arr = obj("data")
    If arr Is Nothing OrElse Not arr.HasValues Then Return Nothing
    Dim d = arr.First
    Return New ResultadoNodo With {
      .CDC = d("cdc").ToString(),
      .Estado = d("estado").ToString(),
      .Aprobado = d("estado").ToString() = "aprobado"
    }
  End Function

  ' ===== CANCELAR DOCUMENTO =====
  Public Shared Async Function CancelarDocumento(cdc As String, motivo As String) As Task(Of Boolean)
    Return (Await Post($"/documentos/{cdc}/cancelar", New With {.motivo = motivo}))("ok").Value(Of Boolean)()
  End Function

  ' ===== DESCARGAR KUDE (PDF) =====
  Public Shared Async Function DescargarKude(cdc As String, Optional fmt As String = "a4") As Task(Of Byte())
    Return Await GetBytes($"/documentos/{cdc}/kude?formato={fmt}")
  End Function

  ' ===== DESCARGAR XML =====
  Public Shared Async Function DescargarXml(cdc As String) As Task(Of Byte())
    Return Await GetBytes($"/documentos/{cdc}/xml")
  End Function

  ' ===== VERIFICAR HEALTH =====
  Public Shared Async Function VerificarHealth(tenantId As String) As Task(Of JObject)
    Return Await [Get]($"/tenants/{tenantId}/health")
  End Function

  ' ===== METODOS INTERNOS =====
  Private Shared Async Function Post(ep As String, payload As Object) As Task(Of JObject)
    Dim c = New StringContent(JsonConvert.SerializeObject(payload), Encoding.UTF8, "application/json")
    Return Await Parse(Await _http.PostAsync(_baseUrl & ep, c))
  End Function

  Private Shared Async Function [Get](ep As String) As Task(Of JObject)
    Return Await Parse(Await _http.GetAsync(_baseUrl & ep))
  End Function

  Private Shared Async Function GetBytes(ep As String) As Task(Of Byte())
    Dim res = Await _http.GetAsync(_baseUrl & ep)
    If Not res.IsSuccessStatusCode Then Throw New Exception("HTTP " & CInt(res.StatusCode))
    Return Await res.Content.ReadAsByteArrayAsync()
  End Function

  Private Shared Async Function Parse(res As HttpResponseMessage) As Task(Of JObject)
    Dim body = Await res.Content.ReadAsStringAsync()
    Dim obj  = JObject.Parse(body)
    If Not res.IsSuccessStatusCode Then
      Throw New Exception("Error API: " & (obj("error")?.ToString() ?? res.StatusCode.ToString()))
    End If
    Return obj
  End Function

End Class

let BASE_URL = '';
let API_KEY  = '';

// ===== INICIALIZACION =====
export function inicializar(baseUrl, apiKey) {
  BASE_URL = baseUrl.replace(/\/$/, '');
  API_KEY  = apiKey;
}

// ===== HELPERS INTERNOS =====
async function request(method, endpoint, body = null) {
  const opts = {
    method,
    headers: {
      'X-API-Key': API_KEY,
      'Content-Type': 'application/json'
    }
  };
  if (body) opts.body = JSON.stringify(body);

  const resp = await fetch(BASE_URL + endpoint, opts);
  const data = await resp.json();

  if (!resp.ok) {
    throw new Error(data.error || `HTTP ${resp.status}`);
  }
  return data;
}

async function requestBinary(endpoint) {
  const resp = await fetch(BASE_URL + endpoint, {
    headers: { 'X-API-Key': API_KEY }
  });
  if (!resp.ok) throw new Error(`HTTP ${resp.status}`);
  return await resp.blob();
}

// ===== VALIDAR DOCUMENTO (sin emitir) =====
export async function validarDocumento(payload) {
  const data = await request('POST', '/documentos/validar', payload);
  return {
    valido: data.valido,
    errores: data.errores || [],
    resumen: data.resumen || null
  };
}

  return data.data;
}

// ===== VERIFICAR WEBHOOK HMAC =====
// Usar solo en el servidor (Node.js), no en el navegador
export async function verificarWebhookHMAC(rawBody, signature, timestamp, secret) {
  const encoder = new TextEncoder();
  const key = await crypto.subtle.importKey(
    'raw', encoder.encode(secret),
    { name: 'HMAC', hash: 'SHA-256' }, false, ['sign']
  );
  const mensaje = encoder.encode(timestamp + '.' + rawBody);
  const sig = await crypto.subtle.sign('HMAC', key, mensaje);
  const hex = Array.from(new Uint8Array(sig))
    .map(b => b.toString(16).padStart(2, '0')).join('');
  return signature === 'sha256=' + hex;
}
